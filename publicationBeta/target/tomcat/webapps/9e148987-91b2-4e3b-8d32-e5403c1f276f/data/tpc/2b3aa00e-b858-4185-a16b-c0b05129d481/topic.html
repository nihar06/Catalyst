The Oracle Home page is displayed.


The Projects window is displayed.


The My Projects page is displayed.


To expand the search area, click the Expand Search button.

To search for the project by project number, click the Project Number field.

Enter all or part of the project number into the Project Number field.


In this example, enter "10009".

To include project status in the search criteria, click the Project Status drop-down list.

To include all statuses in the search, click the All checkbox.

To execute the search for project numbers matching the search criteria, click the Search button.

Project numbers matching the search criteria are listed.


Click to select the appropriate Project Number link.


In this example, click 10009.

The corresponding Overview page is displayed.


The Tasks pane is displayed.


In the Initiate section, click the Manage Financial Project Settings link.

The corresponding Manage Financial Projects Settings page is displayed.


In the Basic Information section, click the Edit button.

The Edit Basic Information window is displayed.


To change the status of the project, click the Change Status button.

The Change Project Status window is displayed.


Click to select the updated status.


In this example, click Approved.

To continue, click the Save and Close button.

To complete the process, click the Save and Close button.

From the Manage Financial Project Settings page, click the Done button.

The Overview page is displayed.


Click the Done button.

Optionally enter appropriate comments into the Comments field.

The project status has been successfully changed without approval.
