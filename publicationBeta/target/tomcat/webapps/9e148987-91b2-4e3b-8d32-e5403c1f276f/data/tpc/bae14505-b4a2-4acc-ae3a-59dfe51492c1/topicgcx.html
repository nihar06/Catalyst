Assigning Assets at the Task Level

The Oracle Home page is displayed.


Click the Projects button.

The Projects window is displayed.


Click the Project Financial Management button.

The My Projects page is displayed.


Assets cannot be assigned at the task level if they are already assigned at the project level and vise versa.


The Search options are displayed.


Enter the appropriate value into the Project Number field.


Enter any additional search criteria as needed.


To execute the search based on the criteria entered, click the Search button.

Project names matching the search criteria are listed.


Click the appropriate project link.

The Basic Information of the corresponding project is displayed.


Click the Manage Financial Project Plan link in the Plan and Execute section.

The Manage Financial Project Plan page is displayed.


The options under Manage Tasks pane is displayed.


The Edit Task Details window is displayed.


Click the Asset Assignments tab.

Select the appropriate Grouping Level Type.


In this example click Specific Assets option.

To add an asset, click the Add option.

All the available assets are displayed in the drop-down list.


Click the Software V2 Asset object.

The Assets are added to the coressponding project.
