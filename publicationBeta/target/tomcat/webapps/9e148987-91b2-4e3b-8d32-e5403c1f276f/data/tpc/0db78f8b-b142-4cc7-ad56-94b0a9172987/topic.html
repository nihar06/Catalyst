Editing a Project Start Date and Project Finish Date

This topic will cover:


- Editing Project Start and End Dates

This topic covered:


- Editing Project Start and End Dates

The Oracle Home page is displayed.


The Projects window is displayed.


The My Projects page is displayed.


To enable search options, click the Expand Search button.

To search by project number, click the Project Number field.

Enter all or part of the project number into the Project Number field.


In this example, enter "D100".

To include project status in the search criteria, click the Project Status drop-down list.

To include all statuses in the search, click the All checkbox.

Enter any additional search criteria as needed.


To execute the search based on the criteria entered, click the Search button.

Projects matching the search criteria entered are listed.


Click to select the project number link corresponding to the project to be updated.


In this example, click the Project Number D10020 link.

The Overview page is displayed.


The Tasks list is displayed.


In the Initiate section, click the Manage Financial Project Settings link.

The Manage Financial Projects Settings page is displayed.


To update the starting and ending dates for the project, click the Edit button.

The Edit Basic Information window is displayed. 


Note: The Start Date or Finish Date may be updated as needed.


To update the ending date of the project, click the Finish Date field.

Click the Clear Date button.

To save the updated information, click the Save and Close button.

From the Manage Financial Project Settings page, click the Done button.

From the Overview page, click the Done button.

The project dates have been updated.

Enter the updated date into the Finish Date field.


In this example, enter "12/31/36".
