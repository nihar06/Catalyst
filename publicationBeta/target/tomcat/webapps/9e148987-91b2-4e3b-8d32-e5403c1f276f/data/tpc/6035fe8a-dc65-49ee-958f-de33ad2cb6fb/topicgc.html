Adding Partner Organization

The Oracle Home page is displayed.


The Projects window is displayed.


The My Projects page is displayed.


To expand the Search area, click the Expand Search button.

To search for the project by project number, click the Project Number field.

Enter all or part of the project number into the Project Number field.

Enter additional search criteria as needed.


To execute the search for project numbers matching the search criteria entered, click the Search button.

Project numbers matching the search criteria are listed.


Click to select the appropriate project number.

The corresponding Overview page is displayed.


The Tasks pane is displayed.


In the Initiate section, click the Manage Financial Project Settings link.

The corresponding Manage Financial Project Settings page is displayed.


The Parties tab is displayed.


Use the scrollbar to navigate to the Partner Organizations section.

In the Partner Organizations section, click the Add button.

The Add Partner Organization window is displayed.


Enter all or part of the partner organization name into the Name field.

To execute the search for organizations matching the search criteria entered, click the Search Name button.

Note:  Partner Organization team members may be included. The Partner Organization Team Member must be associated with the Partner Organization. The names are case sensitive and formatted as First Name Last Name.


To add a partner organization team member, click the Add button.

In the new row, click the Person field.

Enter all or part of the person's name into the Person field.

Note:  If necessary, use the Search... link to search for the appropriate project role.


Click to select the appropriate project role.

Enter the appropriate starting date into the Start Date field.

Note:  If necessary, add the appropriate ending date.


The Overview page is displayed.


Click the Done button.

The partner organization has been successfully added to the project.

Note:  If the partner organization name entered matched only one name, the name defaults. Otherwise, select the appropriate name from the displayed list.
