Creating Nonlabor Costs

The Oracle Home page is displayed.


The Projects window is displayed.


The Overview page is displayed.


The Tasks list is displayed.


In the Capture section, click the Capture Costs link.

The Capture Costs window is displayed.


The Internet Explorer Save window is displayed.


The Connect window is displayed to indicate that you are connecting to the displayed application URL.


To continue, click the Yes button.

The Sign In Oracle Applications window is displayed.


After entering the appropriate user ID and password, click the Sign In button.

The Create Uncosted Nonlabor Expenditure Batch spreadsheet is displayed.


Click to select the appropriate business unit from the list.

Click to select the appropriate document.

Click No.

Enter an appropriate name for the expenditure batch into the Expenditure Batch field.

Enter an appropriate description into the Batch Description field.

In the Expenditure Items section, click the Expenditure Item Date field.

Enter the expenditure date into the Expenditure Item Date field.

Enter the persons name into the Person Name field.

Enter the project name into the Project Name field.

Enter the task name into the Task Name field.

Use the scrollbar to navigate to the Expenditure Type field.

Click the Expenditure Type field.

Enter the expenditure type into the Expenditure Type field.

Enter the nonlabor resource into the Nonlabor Resource field.

Enter the nonlabor resource organization  into the Nonlabor Resource Organization field.

Enter the quantity into the Quantity field.

Click to select the appropriate work type from the list.

Use the scrollbar to navigate back to the beginning of the worksheet.

To close the spreadsheet, click the Close button.

A Microsoft Excel window is displayed to provide the option to save the spreadsheet.


To continue without saving the spreadsheet, click the Don't Save button.

The Export Expenditure Batch to Oracle Fusion Project Costing window is displayed.


To continue, click the Yes button.

The Transactions Export Summary window is displayed to indicate the number of uploaded items.


To continue, click the Close button.

The nonlabor costs have been created.

The Source field is automatically populated.
