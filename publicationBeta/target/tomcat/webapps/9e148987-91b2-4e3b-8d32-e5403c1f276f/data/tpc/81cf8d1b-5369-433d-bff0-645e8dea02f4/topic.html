- Adding a Supplier Organization to a Project

The Oracle Home page is displayed.


The Projects window is displayed.


The My Projects page is displayed.


To expand the Search area, click the Expand Search button.

To search for the project by project number, click the Project Number field.

Enter all or part of the project number into the Project Number field.


In this example, enter "C100".

To include project status as part of the search criteria, click the Project Status drop-down list.

To include all project statuses as part of the search criteria, click the All list item.

To execute the search for projects matching the search criteria entered, click the Search button.

Project numbers matching the search criteria are listed in the Search Results section.


Click to select the appropriate project number.


In this example, click C100.

The corresponding Overview page is displayed.


The Tasks pane is displayed.


In the Initiate section, click the Manage Financial Project Settings link.

The corresponding Manage Financial Project Settings page is displayed.


The Parties tab is displayed.


Click the scrollbar to navigate to the Supplier Organizations section.

In the Supplier Organizations section, click the Add button.

The Add Supplier Organization window is displayed.


Enter all or part of the supplier name into the Name field.


In this example, enter "Staples".

To continue, click the Search Name button.

Supplier names matching the search criteria are listed in the Search Results section.


Click to select the appropriate supplier name.


In this example, click Staples, Inc..

To continue, click the OK button.

Note: The Supplier Organization Team Member must be associated with the Supplier Organization. The names are case sensitive and formatted as First Name Last Name.


To add a team member, click the Add button.

Enter all or part of the person's name into the Person field.


In this example, enter "Ryan Larson".

Note:  If necessary, use the Search... link to search for the appropriate role for the team member.


Click to select the appropriate role for the team member.


In this example, click Project Team Member.

Enter the appropriate starting date into the Start Date field.


In this example, enter "5/1/17".

Note: Use the End Date field as needed.


To continue, click the Save and Close button.

Click the scrollbar to navigate to the Done button.

From the Manage Financial Project Settings page, click the Done button.

The Overview page is displayed.


Click the Done button.

The supplier has been successfully added to the project.
