The Oracle Home page is displayed.


The Projects window is displayed.


The Overview page is displayed.


The Tasks list is displayed.


In the Process section, click the Import Commitments link.

The Import Commitments page is displayed.


Click the Expenditure Business Unit drop-down list.

Click to select the appropriate business unit.


In this example, click Corporate.

Click the Submit button.

A Confirmation message window is displayed to indicate that the process has been submitted.


To continue, click the OK button.

To view the process output, click the Expand Process Monitor button.

Click the scrollbar to navigate to the Process Monitor section.

Note: If the program is not in Succeeded status, click the Refresh button periodically until the status of the program and all child programs are updated to Succeeded.


Click the Refresh button.

Click the View Output button corresponding to the process output you would like to view.


In this example, click the View Output button corresponding to the Import Commitments process.

The corresponding Import Commitments Output window is displayed.


Note: Download the output for better visibility. Validate that there are no errors.


Click the Close button.

Update the Commitment Source Type and Process Mode fields as needed.

Commitments have been successfully imported.
