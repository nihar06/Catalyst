The Oracle Home page is displayed.


The Projects window is displayed.


The My Projects page is displayed.


To expand the search area, click the Expand Search button.

To search for the project by project number, click the Project Number field.

Enter the project number into the Project Number field.


In this example, enter "C100".

To include the project status as search criteria, click the Project Status drop-down list.

To include all statuses in the search criteria, click the All checkbox.

To execute the search for projects matching the search criteria entered, click the Search button.

Project numbers matching the search criteria are listed.


Click to select the appropriate Project Number link.


In this example, click C100.

The corresponding Overview page is displayed.


The Tasks pane is displayed.


In the Initiate section, click the Manage Financial Project Settings link.

The corresponding Manage Financial Project Settings page is displayed.


The Planning tab is displayed.


In the Project Plan Type section, click the Edit button.

The Edit Project Plan Type page is displayed.


Click to select the appropriate calendar type.


In this example, click Accounting calendar.

Note: Update additional fields as needed.


To continue, click the Task Settings link.

The Task Settings tab is displayed.


Note: Update additional fields as needed.


The Currency Settings tab is displayed.


Note: Update additional fields as needed.


The Rate Settings tab is displayed.


Click to select the appropriate resource class raw cost rate value.


In this example, click US Resource Class Cost Rates.

Note: Update additional fields as needed.


The Progress Settings tab is displayed.


Note: Update fields as needed.


The Budget Generation Options tab is displayed.


Click to select the appropriate financial plan type.


In this example, click Approved Cost and Revenue in separate plan version.

Note: Update additional fields as needed.


The Additional Information tab is displayed.


Note: Update fields as needed.


To complete the process, click the Save and Close button.

To continue, click the Done button.

The Overview page is displayed.


Click the Done button.

The project plan type was been successfully edited.
