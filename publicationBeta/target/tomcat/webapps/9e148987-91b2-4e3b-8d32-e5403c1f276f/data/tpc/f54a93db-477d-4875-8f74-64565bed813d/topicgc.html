Generating Allocations

The Oracle Home page is displayed.


The Projects window is displayed.


The Overview page is displayed


The Tasks list is displayed.


In the Allocate section, click the Manage Allocations link.

The Manage Allocations page is displayed.


Click to select the appropriate allocation rule name.

Click to select the appropriate period.

To clear the default date, click the Clear Date button.

Enter the expenditure date into the Expenditure Item Date field.

To submit the process,click the Submit button.

A Confirmation message window is displayed to indicate that the process has been submitted.


To view the status of the submitted process, click the Expand Process Monitor button.

Use the scrollbar to navigate to the Process Monitor section.

Note: If the program is not in Succeeded status, click the Refresh button periodically until the status of the program and all child programs are updated to Succeeded.


Click the Refresh button.

Click the View Output button corresponding to the output you would like to view.

The corresponding Generate Allocations Output page is displayed.


To view the details, click the Default Document link.

Note: Download the output for better visibility.


Validate that there are no errors.


After viewing the report details, click the Close button.

Use the scrollbar to navigate to the Tasks button.

Click the Tasks button.

In the Allocate section, click the Manage Allocations link.

The Manage Allocations page is displayed.


In the Search section, click the Business Unit drop-down list.

Click to select the appropriate business unit name.

Click to select the appropriate allocation rule name.

To execute the search for allocation rules matching the search criteria entered, click the Search button.

Matching allocation rule names are listed.


Click to select the appropriate rule.

After selecting the allocation rule, click the Actions menu.

A Warning message window is displayed to indicate that the allocation will be released.


To continue, click the Yes button.

An Information message window is displayed to indicate that the process has been submitted.


To continue, click the OK button.

To complete the process, click the Done button.

Use the scrollbar to navigate to the Process Monitor section.

Click the View Output button corresponding to the output you would like to view.

Note: Download the output for better visibility.


Validate that there are no errors.


To close the window, click the Close button.

The Manage Allocations tab is displayed.


In the Parameters section, the Business Unit field is automatically populated and may be updated if needed.

Note: If the program is not in Succeeded status, click the Refresh button periodically until the status of the program and all child programs are updated to Succeeded.


Click the Refresh button.

The allocations have been generated successfully.
