Select the desired Playback Mode from the toolbar above.  Descriptions of the playback modes have been provided below.

  • The Try It! mode enables users to learn a task by practicing in a simulated environment with step-by-step instructions and guidance throughout the task.  If you make a mistake, you will be prompted to try again.
     
  • The Do It! mode enables users to be guided through a transaction while performing the steps in a live environment (sandbox or production).  This unique mode makes it possible for you to accomplish real work while you learn.
