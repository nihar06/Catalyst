- Editing Task Level Transactional Controls

The Oracle Home page is displayed.


The Projects window is displayed.


The My Projects page is displayed.


To expand the search area, click the Expand Search button.

To search for the project by project number, click the Project Number field.

Enter all or part of the project number into the Project Number field.


In this example, enter "C100".

To execute the search for project numbers matching the search criteria, click the Search button.

Project numbers matching the search criteria are listed.


Click to select the appropriate Project Number link.


In this example, click the C100 link.

The corresponding Overview page is displayed.


The Tasks pane is displayed.


In the Plan and Execute section, click the Manage Financial Project Plan link.

The corresponding Manage Financial Project Plan page is displayed.


Click the Transaction Controls button corresponding to the Document Control task.

The Edit Transaction Controls page is displayed.


Click to select the appropriate transaction control option.


In this example, click Exclusive.

To add a new control, click the Add Row button.

In the new row, click the Expenditure Category drop-down list.

To search for the appropriate expenditure category, click the Search... link.

The Search and Select: Expenditure Category window is displayed.


Enter all or part of the category name into the Name field.


In this example, enter "Technology".

To execute the search for category names matching the search criteria entered, click the Search button.

Category names matching the search criteria are listed.


Click to select the appropriate category from the listing.


In this example, click Technology.

Click to select the appropriate expenditure type from the list.


In this example, click Hardware.

Click the scrollbar to navigate to additional fields.

Click to select the appropriate billable option.


In this example, click No.

To change the default starting date, click the From Date field.

Enter the appropriate starting date into the From Date field.


In this example, enter "1/1/17".

Note: Update any additional fields as needed.


To complete the process, click the Save and Close button.

From the Manage Financial Project Plan page, click the Save and Close button.

The Overview page is displayed.


Click the Done button.

The Task Level Transaction Controls have been successfully edited.

Note: Transaction controls can be set at the project level and/or task level. The task level transactions controls takes precedence over the project level transactions controls.
