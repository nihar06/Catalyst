- Creating a Project From Another Project

The Oracle Home page is displayed.


The Projects window is displayed.


Click the Project Financial Management button.

The My Projects page is displayed.


Click the Create Project drop-down list.

Click the From Project command.

The Create Project from Another Project page is displayed.


To search for the required Contract, Capital, Capital Billable or Indirect Project to be used as the copy, click the Project Number field.

Enter all or part of the appropriate Contract, Capital, Capital Billable or Indirect Project number into the Project Number field.


In this example, enter "100021".

To execute the search based on the criteria entered, click the Search button.

Project numbers matching the search criteria are listed in the Search Results section.


Click to select the appropriate Project Name cell.


In this example, click Billable Project 100021.

To generate the new project from the selected project, click the Create from Project button.

The Create Project page is displayed.


Note: The Project Name and Project Number values must be unique.


Enter the project name into the Project Name field.


In this example, enter "Billable Project 100022".

Enter the project number into the Project Number field.


In this example, enter "100022".

Click the Project Start Date field.

To delete the default date, click the Clear Date button.

Enter the appropriate starting date into the Project Start Date field.


In this example, enter "5/15/17".

Enter the appropriate project ending date into the Project Finish Date field.


In this example, enter "12/31/20".

To include project manager information, click the Search: Project Manager button.

The Search and Select: Project Manager window is displayed.


Click the Name field.

Enter all or part of the project manager's name into the Name field.


In this example, enter "Marlin".

Enter any additional search criteria as needed.


To execute the search based on the criteria entered, click the Search button.

Names matching the search criteria entered are listed.


Double-click to select the appropriate name.


In this example, double-click Marlin, Amy.

Click the scrollbar to navigate to the Copy Options section.

Click the scrollbar to navigate to the Financial Plans section.

Click the scrollbar to navigate up to the Save and Continue button.

In the General tab, project general details are displayed.


Click the Parties link.

In the Parties tab, all valid team members, project customers, partner organizations and supplier organizations are listed.


In the Planning tab, additional project planning details are displayed.


In the Financial Tab, project financial details are displayed.


In the Reporting tab, reporting details are displayed.


To continue, click the Done button.

From the Overview page, click the Done button.

The Organization and Legal Entity values default based on the template selected.


The fields may be updated as needed.

The Quick Entry fields defined on a project template control the minimum required information to create a project using that template.


For example, apart from the default required fields such as Project Name and Project Number, the project template may require that you specify a project manager and project administrator, or a funding source.

In the Copy Options section, use the checkboxes next to the Project Information and Project Plan Information fields to copy the details from the selected template.


In this example, accept the defaults.

In the Financial Plans section, use the checkboxes next to the Financial Plans and Other Financial Information fields to copy the information from the selected template.


In this example, accept the defaults.

The project has been successfully created from another project.

The Manage Financial Project Settings page is displayed.


The project definitions are addressed in detail on the referenced tabs of this worksheet.
