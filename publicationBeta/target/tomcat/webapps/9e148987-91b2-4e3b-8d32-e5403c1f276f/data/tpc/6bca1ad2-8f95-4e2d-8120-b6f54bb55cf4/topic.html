Creating a Billable Project from Template

This topic will cover:


- Creating a Billable Project from a Template

This topic covered:


- Creating a Billable Project from a Template

The Oracle Home page is displayed.


The Projects window is displayed.


The My Projects page is displayed.


Click From Template.

The Create Project from Template page is displayed.


Enter all or part of the template name to be used into the Template Name field.


In this example, enter "US PS Billable".

To execute the search for template names matching the search, click the Search button.

Template names matching the search criteria entered are listed in the Search Results section.


Click to select the appropriate template name link to be used.


In this example, click US PS Billable no Burden.

To create a new project from the selected template, click the Create from Template button.

The Create Project page is displayed.


Note: The Project Name and Project Number must be unique values.


Enter an appropriate project name into the Project Name field.


In this example, enter "Billable Project 100021".

Enter the appropriate project number into the Project Number field.


In this example, enter "100021".

To delete the existing project starting date, click the Clear Date button.

Enter the appropriate starting date into the Project Start Date field.


In this example, enter "5/1/17".

Enter the appropriate project ending date into the Project Finish Date field.


In this example, enter "12/31/20".

To identify the project manager, click the Search: Project Manager button.

The Search and Select: Project Manager page is displayed.


Enter all or part of the project manager's name into the Name field.


In this example, enter "Marlin".

To execute the search for names matching the search criteria entered, click the Search button.

Names matching the search criteria are listed.


Double click to select the appropriate name from the list.


In this example, double-click Marlin, Amy.

Click the scrollbar to navigate to the Copy Options section.

Click the scrollbar to navigate to the Financial Plans section.

Click the scrollbar to navigate up to the Save and Continue button.

Click the Parties link.

The Parties tab is displayed.


All valid team members, project customers, partner organizations and supplier organizations are listed.


The Planning tab is displayed.


Additional project planning details are displayed.


The Financial tab is displayed.


Project financial details are displayed.


The Reporting tab is displayed.


Project reporting details are displayed.


To continue, click the Done button.

From the Overview page, click the Done button.

The Organization and Legal Entity values default based on the template selected.


The fields may be updated as needed.

The Quick Entry fields defined on a project template control the minimum required information to create a project using that template.


For example, apart from the default required fields such as Project Name and Project Number, the project template may require that you specify a project manager and project administrator, or a funding source.

In the Copy Options section, use the checkboxes next to the Project Information and Project Plan Information fields to copy the details from the selected template.


In this example, accept the defaults.

In the Financial Plans section, use the checkboxes next to the Financial Plans and Other Financial Information fields to copy the information from the selected template.


In this example, accept the defaults.

The billable project has been successfully created from a template.

The Manage Financial Project Settings page is displayed.


The project definitions are addressed in detail on the referenced tabs of this page.
