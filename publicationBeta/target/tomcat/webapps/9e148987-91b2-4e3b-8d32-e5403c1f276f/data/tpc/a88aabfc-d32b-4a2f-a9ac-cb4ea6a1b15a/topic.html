This topic covered:


- Adding Team Members to a Project

The Oracle Home page is displayed.


The Projects window is displayed.


The My Projects window is displayed.


To expand the search area, click the Expand Search button.

To search for the project by project number, click the Project Number field.

Enter all or part of the project number into the Project Number field.


In this example, enter "D1000".

To include project status in the search criteria, click the Project Status drop-down list.

To include all statuses in the search criteria, click the All list item.

Enter any additional search criteria as needed.


To execute the search based on the criteria entered, click the Search button.

Project numbers matching the search criteria are listed.


Click to select the appropriate project number link.


In this example, click D10000.

The corresponding Overview page is displayed.


The Tasks list is displayed.


In the Initiate section, click Manage Financial Project Settings.

The corresponding Manage Financial Project Settings page is displayed.


Click the Parties tab.

In the Team Members section, click the Add... button.

The Add Team Members window is displayed.


Note: The names are case sensitive and formatted "Last Name, First Name".


To search for the employee name to be added to the project, enter all or part of the employee's name into the Person field.


In this example, enter "Brown, Casey".

Note:  If necessary, use the More... link to search for the employee name to be added.


Click to select the employee name.


In this example, click Brown, Casey Director of IT (jgorirossi@kbace.com).

To include a starting date, click the Start Date field.

Enter the appropriate starting date into the Start Date field.


In this example, enter "5/1/17".

To indicate the project role for the employee, click the Project Role drop-down list.

Note:  If necessary, use the Search... link to search for the appropriate project role.


Click to select the appropriate role from the list.


In this example, click Project Team Member.

To continue, click the Save and Close button.

From the Manage Financial Project Settings page, click the Done button.

From the Overview page, click the Done button.

Use the End Date field as needed.

Optionally, use the Track Time checkbox to track a team member's timecard entry for project performance reporting.

Use the Save and Add Another button to add another team member if necessary.

The team member has been successfully added to the project.
