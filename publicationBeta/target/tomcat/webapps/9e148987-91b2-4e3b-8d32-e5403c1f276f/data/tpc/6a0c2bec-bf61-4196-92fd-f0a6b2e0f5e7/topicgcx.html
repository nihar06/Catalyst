Creating Contract Lines and Fund Project

The Oracle Home page is displayed.


The Contract Management window is displayed.


The Contracts page is displayed.


In the Contract Lists section, click the Expand Search button.

In the expanded Search section, click the Number field.

Verify the operator value to be used.  Update as needed using the drop-down list.


Enter the appropriate contract number into the Number field.

To search for the contract by status, click the Status drop-down list.

To search contracts with any status, click All.

Use the scrollbar to navigate to the Search button.

To execute the search for contracts matching the search criteria entered, click the Search button.

Matching contracts are listed in the Search Results section.


Click the Number link corresponding to the appropriate contract.

The Corresponding Edit Contract: Over page is displayed.


The Lines tab is displayed.


The Create Line window is displayed.


Enter a line number into the Number field.

Click to select the appropriate type from the list.

Enter contract line name into the Name field.

To delete the default starting date, click the Clear Date field.

Enter the appropriate contract line starting date into the Start Date field.

Note: Use the Create Another button as needed to create another line as needed.


To continue, click the OK button.

In the Line 1: Details section, click the Description field.

Enter an appropriate description into the Description field.

Use the scrollbar to navigate to the Accounts section.

The Associated Projects tab was displayed.


In the new row, click the Project Name drop-down list.

Click to select the appropriate project name from the list.

Enter the appropriate funded amount for the line into the Funded Amount field.

To save the contract line details, click the Save menu.

To close the window, click the Save and Close command.

Enter any additional field values as needed.

In the Accounts section, use additional fields as needed.

Optionally use the Task Name field.

Add additional project or project/task associations to the selected line as needed.

The contract line has been created and funding amounts have been successfully added.
