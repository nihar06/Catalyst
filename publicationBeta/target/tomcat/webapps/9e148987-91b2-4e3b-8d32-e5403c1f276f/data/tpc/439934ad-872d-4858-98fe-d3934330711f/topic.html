The Oracle Home page is displayed.


Click the Projects button.

The Projects window is displayed.


Click the Costs button.

The Overview page is displayed.


Click the Tasks button.

The Tasks list is displayed.


In the Process section, click the Import Costs link.

Click Oracle Fusion Projects.

Click Time Card.

Click the scrollbar to view additional fields.

Click to select the appropriate report option.


In this example, click Detail.

A Confirmation message window is displayed to indicate that the process has been submitted.


To view the status of the submitted process, click the Expand Process Monitor button.

Click the scrollbar to navigate to the Process Monitor section.

Note: If the program is not in Succeeded status, click the Refresh button periodically until the status of the program and all child programs are updated to Succeeded.


To view the report details, click the View Output button corresponding to the Import Costs: Generate Output Report process name.

The Import Costs: Generate Output Report window is displayed.


Click the report link to view the output.


In this example, click the ESS_O_152411 link.

Note: Download the output for better visibility.


To close the report window, click the Close button.

The Import Costs page is displayed.


The Business Unit, Process Mode and Transaction Status fields are displayed by default and may be updated as needed.

The Process Through Expenditure Item Date field defaults to today's date and may be updated as needed.

The Import Costs Output Report window is displayed.


Note: Validate that there are no errors.


Click the Close button.

The Projects - Time Card Costs have been successfully imported and processed.
