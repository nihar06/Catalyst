Creating a Contract

The Oracle Home page is displayed.


The Contract Management window is displayed.


The Contracts page is displayed.


The Tasks list is displayed.


In the Contracts section, click the Create Contract link.

Click to select the appropriate contract type.

Click to select the appropriate primary party.

To delete the default date, click the Clear Date button.

Enter the appropriate starting date into the Start Date field.

To continue, click the Save and Continue button.

Enter the appropriate contract name into the Name field.

In the Tax Control section, click the Classification Code drop-down list.

Click to select the appropriate tax classification code.

The Create Contact window is displayed.


Enter the contact's first name into the First Name field.

Enter the contact's last name into the Last Name field.

Click the Area Code field of the Phone number.

Enter the appropriate area code into the Area Code field.

Click the Phone Number field.

Enter the phone number into the Phone field.

Add additional contact phone numbers as needed.


Enter the contact's email address into the E-Mail field.

Enter address line details into the Address Line 1 field.

Enter additional address line details as needed.


Enter the address postal code into the Postal Code field.

If the postal code entered matches multiple city, state and county combinations, click to select the appropriate combination.


To add a contact role, click the Add button.

In the new record, click the Role drop-down list.

Click to select the appropriate role.

Enter the contact's name into the Name field.

Use the scrollbar to navigate to the Accounts section.

In the Accounts section, click the Bill-to Account Number drop-down list.

Click to select the appropriate account number.

Click to select the appropriate bill-to site.

Use the scrollbar to navigate to the Save button.

To save the information entered, click the Save button.

In the Contract Documents section, click the Actions menu.

Click Add.

In the new row, click the Category drop-down list.

Click to select the appropriate category.

The Choose File to Upload window is displayed.


After navigating to the appropriate folder, click to select the appropriate contract document.

To continue, click the Open button.

Enter an appropriate title into the Title field.

Enter an appropriate description into the Description field.

To save the contract document details entered, click the Save menu.

To close the page, click the Save and Close button.

The Create Contract window is displayed.


Update the Business Unit and Legal Entity fields as needed.

Update the Currency field as needed.

The corresponding Edit Contract: Overview page is displayed.


The Contract Number may generate automatically if enabled during your implementation.


If necessary, use the Number field to enter a contract number.

Enter an appropriate description into the Description field.

Verify the Start Date and End Date fields.

The Parties tab is displayed.


Verify that the Supplier role is the Business Unit selected during contract creation.

The Team Members section toggles to the Contacts section, depending on the Party Role selected.


If a customer contact cannot be found, use the Create Contact button.

To continue, click the Save and Close button.

Contract terms will default from the contract terms template.


Optionally, run the contract expert to add additional clauses and variables.

The contract has been created successfully.
