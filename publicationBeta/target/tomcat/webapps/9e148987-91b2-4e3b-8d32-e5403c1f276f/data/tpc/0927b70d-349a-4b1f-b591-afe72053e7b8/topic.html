- Managing Events

The Oracle Home page is displayed.


The Contract Management window is displayed.


The Invoices page is displayed.


The Tasks list is displayed.


In the Review and Adjust section, click the Manage Events link.

The Manage Events page is displayed.


To search by event type, click the Event Type drop-down list.

Click to select the appropriate event type.


In this example, Click the Fee item.

Enter any additional search criteria as needed.


To execute the search based on the criteria entered, click the Search button.

Event types matching the search criteria are listed.


Click the appropriate event number link.


In this example, click the 1 number link.

Click the scrollbar to navigate to the Status section

Click the Expand Status button.

To place an hold on an event, click the Hold Invoice drop-down list.

Click to select the hold invoice option.


In this example, click Once.

Click the scrollbar to navigate to the Save and Close button.

Click the Save and Close button.

From the Manage Events page, click the Done button.

The corresponding Edit Event page is displayed.


Review the event.


If needed, update the Description field or Amount in Bill Transaction Currency field if the event has yet to be processed.

Managing an event has been successfully completed.
