- Creating a Contract With the Contract Wizard

This topic covered:


- Creating a Contract With the Contract Wizard

The Oracle Home page is displayed.


The Contract Management window is displayed.


The Contracts page is displayed.


The Tasks list is displayed.


In the Contracts section, click the Create Contract in Wizard link.

Click to select the appropriate contract type.


In this example, click Sell: Project Lines Soft Limit.

Click to select the appropriate primary party for the contract.


In this example, click Citi.

To delete the default date, click the Clear Date field.

Enter the contract starting date into the Start Date field.


In this example, enter "01/01/17".

To continue, click the Save and Continue button.

Enter an appropriate name into the Name field.


In this example, enter "Citi Contract".

Enter the appropriate value into the Description field.


In this example, enter "Citi Application Implementation".

In the Tax Control section, click the Classification Code drop-down list.

Click to select the appropriate classification code from the list.


In this example, click NL VAT STANDARD RATE.

In the Contacts section, click the Add button to add a contact.

In the new record, click the Role drop-down list.

Click to select the appropriate contact role.


In this example, click  Contract administrator.

Enter the contact's name into the Name field.


In this example, enter "Dean Larson".

Click the scrollbar to navigate to the Accounts section.

In the Accounts section, click the Bill-to Account Number drop-down list.

Click to select the appropriate account number.


In this example, click 3001.

Click to select the appropriate bill-to site.


In this example, click 45006.

Click the scrollbar to navigate to the Save button.

To save the contract details, click the Save button.

The Contract Documents tab is displayed.


In the new row, click the Category drop-down list.

Click Contract.

Click the File Name or URL field.

The Choose File to Upload window is displayed


After navigating to the appropriate folder, click to select the associated document.


In this example, click Citi Contract.pdf.

To continue, click the Open button.

Optionally, enter a document title into the Title field.


Enter an appropriate description into the Description field.


In this example, enter "Contract".

To save the attached documents, click the Save menu.

To close the window, click the Save and Close button.

The Create Contract in Wizard window is displayed.


Update the Business Unit and Legal Entity field values as needed.

Update the Currency field as needed.

The corresponding Edit Contract: Overview page is displayed.


The Contract Number may generate automatically if enabled during your implementation.


If necessary, use the Number field to enter a contract number.

Verify the Start Date and End Date field values. Update the dates as needed.

The Parties tab is displayed.


Verify that the Supplier role is the Business Unit selected during contract creation.

The Team Members section toggles to the Contacts section, depending on the Party Role selected.


If a customer contact cannot be found, use the Create Contact button.

Contract terms will default from the contract terms template.


Optionally, run contract expert to add additional clauses and variables.

The contract has been created successfully from the contract wizard.
