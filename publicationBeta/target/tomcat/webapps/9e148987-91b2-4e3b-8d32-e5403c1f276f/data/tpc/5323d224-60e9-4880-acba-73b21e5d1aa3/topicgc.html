Creating Nonlabor Costs for Third-Party Applications

The Oracle Home page is displayed.


The Projects window is displayed.


The Overview page is displayed.


The Tasks list is displayed.


In the Capture section, click the Capture Costs link.

The Capture Costs window is displayed.


The Internet Explorer Save window is displayed.


The Connect window is displayed to indicate that you will connect to the displayed application URL.


The Sign In Oracle Applications window is displayed.


After entering the appropriate user ID and password, click the Sign In button.

The Create Uncosted Nonlabor Expenditure Batch for Third-Party Applications spreadsheet is displayed.


Click to select the appropriate business unit from the list.

Click to select the appropriate source from the list.

Enter the expenditure batch name into the Expenditure Batch field.

Enter an appropriate description into the Batch Description field.

In the Expenditure Items section, click the Document Entry field.

Enter the appropriate document entry value into the Document Entry field.

Enter the expenditure item date into the Expenditure Item Date field.

Enter the person's name into the Person Name field.

Enter the project name into the Project Name field.

Use the scrollbar to navigate to the Task Name field.

Click the Task Name field.

Enter the appropriate task name into the Task Name field.

Enter the expenditure type into the Expenditure Type field.

Enter the nonlabor resource into the Nonlabor Resource field.

Click the Nonlabor Resource Organization field.

Enter the resource into the Nonlabor Resource Organization field.

Enter the quantity into the Quantity field.

Click to select the appropriate work type.

Use the scrollbar to navigate to the Original Transaction Reference field.

Click the Original Transaction Reference field.

Enter a transaction reference into the Original Transaction Reference field.

Use the scrollbar to navigate back to the beginning of the worksheet.

The Export Expenditure Batch to Oracle Fusion Project Costing window is displayed.


To continue, click the OK button.

The Transactions Export Summary window is displayed to indicate the number of items uploaded.


To close the window, click the Close button.

To close the spreadsheet, click the Close button.

The Microsoft Excel window is displayed to provide the option to save the spreadsheet.


To continue without saving the spreadsheet, click the Don't Save button.

The Document field is automatically populated.

The nonlabor costs have been created for the third-party applications.
