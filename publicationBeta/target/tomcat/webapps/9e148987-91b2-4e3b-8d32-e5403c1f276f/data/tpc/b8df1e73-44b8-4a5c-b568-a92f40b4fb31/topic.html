The Oracle Home page is displayed.


Click the Projects button.

The Projects window is displayed.


Click the Costs button.

The Costs page is displayed.


The Tasks list is displayed.


In the Capture section, click the Manage Unprocessed Costs link.

The Manage Unprocessed Costs page is displayed.


To search by project number, click the Project Number field.

Enter the appropriate number into the Project Number field.


In this example, enter "C032817".

Enter additional search criteria as needed.


To execute the search based on the criteria entered, click the Search button.

Click the scrollbar to navigate to the Search Results section.

Projects matching the search criteria are listed.


Click to select the row corresponding to the appropriate project.


In this example, click the row corresponding to Contract C032817.

With the appropriate row selected, click the Actions menu.

Click the Edit command.

Click the scrollbar to view the General Details tab.

Click the scrollbar to navigate up to the Save menu.

Click the Save menu.

Click the Save and Close command.

In the Search Results section of the Manage Unprocessed Costs page, click the Import and Process button.

A Warning message window is displayed to indicate that you have selected to import and process the cost transactions.


Click the Yes button.

A Confirmation message window is displayed to indicate that the process request has been submitted.


Click the OK button.

Click the Overview tab.

Click the Expand Process Monitor button.

Click the scrollbar to navigate to the Process Monitor section.

If the program is not in Succeeded status, click the Refresh button periodically until the status of the program and all child programs are updated to Succeeded.


Click the View Output button corresponding to the process you would like to view.


In this example, click the View Output button corresponding to Process ID 146763.

An Output message window is displayed.


Click the output name link to view the output.


In this example, click ESS_O_146763.

To download the output for better visibility,click the Open button.

Validate that there are no errors.


Update all the values that are in error.

The unprocessed costs has been managed successfully.
