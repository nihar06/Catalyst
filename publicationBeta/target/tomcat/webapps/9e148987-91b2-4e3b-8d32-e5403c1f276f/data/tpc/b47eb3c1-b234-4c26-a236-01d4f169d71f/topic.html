The Oracle Home page is displayed.


The Projects window is displayed.


The My Projects page is displayed.


To expand the Search sections, click the Expand Search button.

The Search section is displayed.


To search by project number, click the Project Number field.

Enter the appropriate project number into the Project Number field.


In this example, enter "C100000".

To execute the search based on the criteria entered, click the Search button.

Projects matching the search criteria are listed.


Click to select the appropriate project number link.


In this example, click Capital 100000.

The corresponding Overview page is displayed.


The Tasks pane is displayed.


In the Control section, click the Manage Project Budget link.

The corresponding Manage Budget Versions page is displayed.


The Create New Version: Manage Version Details page is displayed.


Click to select the appropriate financial plan type.


In this example, click Approved Cost and Revenue in separate plan version.

Click to select the appropriate budget creation method.


In this example, click Manually create budget lines.

In the Budget Creation Details section, click the Show More link.

The Plan Settings tab is displayed.


Note: Review or edit the details as needed.


The Generation Options tab is displayed.


Click to select the appropriate generation source.


In this example, click Financial plan type.

Click the scrollbar to navigate to additional fields.

Click to select the appropriate source plan version.


In this example, click Approved Cost and Revenue in separate plan version.

Click to select the appropriate source plan version.


In this example, click Version 1.

Note: Review or update the additional tabs as needed.


The Create New Version: Edit Budget page is displayed.


Click the Expand button corresponding to the appropriate task number or resource to view the sub tasks.


In this example, click the Expand button corresponding to C100000.

Click to select the appropriate task number or resource to be updated.


In this example, click 1.0.

The Assign Financial Resources to Selected Tasks window is displayed.


Click to select the appropriate resource.


Note: Press the CTRL key and click to select rows randomly or press the Shift key and click to select continuous rows.


In this example, click Financial Resources.

Click the Expand button corresponding to the appropriate task number or resource to be updated.


In this example, click the Expand button corresponding to 01.

The corresponding resource is displayed.


Click the Raw Cost field corresponding to the appropriate resource.


In this example, click the Raw Cost field corresponding to Financial Resources.

Enter the appropriate cost into the Raw Cost field.


In this example, enter "500000".

Note: Update additional resources as needed.


The Create New Version: Review and Finalize Budget page is displayed.


The Submit Version message window is displayed.


To continue, click the OK button.

The Manage Budget Versions page is displayed.


The Navigator is displayed.


Click the scrollbar to navigate to the More... link.

Click the More... link.

The Scheduled Processes page is displayed.


The Schedule New Process window is displayed.


The Search and Select: Name window is displayed.


Enter all or part of the appropriate name into the Name field.


In this example, enter "Update Project Performance Data".

To execute the search, click the Search button.

Processes matching the search criteria are listed.


Click to select the appropriate process.


In this example, click Update Project Performance Data Without Producing Report.

The Schedule New Process window is displayed.


The Process Details page is displayed.


To search by business unit, click the Search: Business Unit button.

The Search and Select: Business Unit window is displayed.


Enter all or part of the appropriate business unit into the Business Unit field.


In this example, enter "Corporate".

To execute the search based on the criteria entered, click the Search button.

Business units matching the search criteria are listed.


Double-click to select the appropriate business unit.


In this example, double-click Business Unit.

Note: Update any additional fields as needed.


Click the Submit button.

The Confirmation message window is displayed to indicate the process was submitted.


Verify that the update project performance data without producing report process has completed successfully and the status shows Succeeded.

Budgeting for budgetary control has been successfully completed.

Use the Notes and Attachments buttons as needed.
