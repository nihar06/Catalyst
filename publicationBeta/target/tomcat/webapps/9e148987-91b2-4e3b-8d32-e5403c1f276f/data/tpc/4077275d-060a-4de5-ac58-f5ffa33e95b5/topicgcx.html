Creating Labor Costs

The Oracle Home page is displayed.


The Projects window is displayed.


The Overview page is displayed.


The Tasks list is displayed.


In the Capture section, click the Capture Costs link.

The Capture Costs window is displayed.


The Internet Explorer Save window is displayed.


Click the Open option.

A Connect message window is displayed to indicate that you will connect to the displayed application URL.


To continue, click the Yes button.

The Sign In Oracle Applications window is displayed.


After entering the appropriate user ID and password, click the Sign In button.

The Create Uncosted Labor Expenditure Batch spreadsheet is displayed.


To select the business unit, click the Business Unit drop-down list.

Click to select the appropriate business unit from the list.

Enter the expenditure batch name into the Expenditure Batch field.

Enter an appropriate description of the batch into the Batch Description field.

Enter the document entry information into the Document Entry field.

Enter the appropriate date into the Expenditure Item Date field.

Enter the person's name into the Person Name field.

Enter the project name into the Project Name field.

Use the scrollbar to navigate to the Task Name field.

Click the Task Name field.

Enter the task name into the Task Name field.

Enter the expenditure type into the Expenditure Type field.

Enter the appropriate quantity into the Quantity field.

Click to select the appropriate work type. 

Use the scrollbar to navigate back to the beginning of the worksheet.

To export the spreadsheet data, click the Export button.

The Export Expenditure Batch to Oracle Fusion Project Costing window is displayed to indicate that the expenditure batch will be exported to Oracle Fusion Project Costing and that the details may be reviewed and released after exporting.


To continue, click the Yes button.

The Transaction Export Summary window is displayed to indicate the number of uploaded items.


To close the window, click the corresponding Close button.

To close the spreadsheet, click the Close button.

A Microsoft Excel window is displayed to provide the option of saving the spreadsheet.


To continue without saving the spreadsheet, click the Don't Save button.

The Source and Document fields are automatically populated.

The labor costs have been successfully created.
