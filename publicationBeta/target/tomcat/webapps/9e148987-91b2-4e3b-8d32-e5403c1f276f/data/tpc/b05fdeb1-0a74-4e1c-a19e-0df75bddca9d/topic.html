- Approving a Project Forecast

The Oracle Home page is displayed.


The Projects window is displayed.


The My Projects page is displayed.


Click the Project Number link corresponding to the project forecast to be approved.


In this example, click C10009.

The corresponding Overview page is displayed.


The Tasks pane is displayed.


In the Control section, click Manage Project Forecast.

The corresponding Manage Forecast Versions page is displayed.


Click to select the version Name cell corresponding to the version to be reviewed and approved.


In this example, click Version 1.

Click Review and Finalize Forecast.

The corresponding Edit Version: Review and Finalize Forecast page is displayed.


Note: If the forecast is ready to send for approval use the Submit menu and then click the Create Baseline command.


The corresponding Submit Version window is displayed.


To continue, click the OK button.

From the Overview page, click the Done button.

The project forecast has been successfully approved.
