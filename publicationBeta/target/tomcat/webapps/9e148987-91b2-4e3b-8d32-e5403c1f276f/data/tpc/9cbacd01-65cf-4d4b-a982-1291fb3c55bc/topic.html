The Oracle Home page is displayed.


The Projects window is displayed.


The Overview page is displayed.


The Tasks list is displayed.


In the Capture section, click the Capture Costs link.

The Capture Costs window is displayed.


The Create Labor Costs for Third-Party Applications spreadsheet is displayed.


Click to select the appropriate business unit from the list.


In this example, click Corporate.

Click to select the appropriate source from the list.


In this example, click External Time Entry System.

Enter the expenditure batch name into the Expenditure Batch field.


In this example, enter "Labor123h".

Enter an appropriate description into the Batch Description field.


In this example, enter "Labor123h".

In the Expenditure Items section, click the Document Entry field.

Enter the appropriate document entry into the Document Entry field.


In this example, enter "Straight Time".

Click the Expenditure Item Date field.

Enter the item expenditure date into the Expenditure Item Date field.


In this example, enter "9/7/2017".

Enter the person's name into the Person Name field.


In this example, enter "Marlin, Amy".

Enter the project name into the Project Name field.


In this example, enter "Contract C032817".

Click the scrollbar to navigate to the Task Name field.

Enter the task name into the Task Name field.


In this example, enter "Expenses".

Enter the expenditure type into the Expenditure Type field.


In this example, enter "Administrative".

Enter the appropriate quantity into the Quantity field.


In this example, enter "40".

Click to select the appropriate work type.


In this example, click Billable.

Enter the appropriate reference information into the Original Transaction Reference field.


In this example, enter "Time Card 123x".

Click the scrollbar to navigate back to the beginning of the worksheet.

The Export Expenditure Batch to Oracle Fusion Project Costing window is displayed.


Click the scrollbar to navigate to the OK button.

To continue, click the OK button.

The Transactions Export Summary window is displayed to indicate the number of uploaded items.


To continue, click the Close button.

To close the spreadsheet, click the Close button.

A Microsoft Excel window is displayed to provide the option to save the spreadsheet.


To close the spreadsheet without saving, click the Don't Save button.

To close the Capture Costs window, click the Close button.

The Internet Explorer Save window is displayed.


The Sign In Oracle Applications window is displayed.


After entering the appropriate user ID and password, click the Sign In button.

The Connect window is displayed to indicate that you will connect to the displayed application URL.


To continue, click the Yes button.

The Document field is automatically populated.

The labor costs for a third party application have been created.
