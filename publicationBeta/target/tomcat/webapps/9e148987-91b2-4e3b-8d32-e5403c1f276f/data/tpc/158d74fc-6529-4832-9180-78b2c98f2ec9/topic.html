The Oracle Home page is displayed.


Click the Projects button.

The Projects window is displayed.


Click the Project Financial Management button.

The My Projects page is displayed


Click to select the appropriate project number link.


In this example, click D10009.

The corresponding Overview page is displayed.


Click the Tasks button.

The Tasks pane is displayed.


In the Control section, click Manage Project Forecast.

The corresponding Manage Budget Versions page is displayed.


The Manage Forecast Versions page is displayed.


Click the Done button.

The corresponding Edit Version: Edit Forecast page is displayed.


If the forecast is ready to send for approval, click Submit > Create Baseline > OK.


If you selected Create Baseline in error, select Cancel to return to the Edit Forecast page.


If the forecast is to be revised, click Back and return to the Edit Forecast page.


If the forecast is to be completed at a later time, click Save > Save and Close.

The Overview page is displayed.


Click the Done button.

The forecast has been successfully unlocked.
