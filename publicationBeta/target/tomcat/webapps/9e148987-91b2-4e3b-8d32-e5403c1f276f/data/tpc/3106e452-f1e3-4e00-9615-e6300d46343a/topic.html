Creating a Overhead Project from Template

This topic will cover:


- Creating an Overhead Project from a Template

This topic covered:


- Creating an Overhead Project from a Template

The Oracle Home page is displayed.


The Projects window is displayed.


The My Projects page is displayed.


Click the drop-down arrow to the right of the Create Project box.

Click the From Template command.

The Create Project from Template page is displayed.


Enter all or part of the project template name into the Template Name field.


In this example, enter "US Internal".

Enter any additional search criteria as needed.


To execute the search based on the criteria entered, click the Search button.

Project template names matching the search criteria are listed.


Click to select the row corresponding to the appropriate project name.


In this example, click the row corresponding to US Internal Capital no Burden.

After selecting the appropriate template, click the Create from Template button.

The Create Project page is displayed.


Note: The Project Name and Project Number must be unique values.


Enter an appropriate name for the project into the Project Name field.


In this example, enter "Internal 10008".

Enter the appropriate project number into the Project Number field.


In this example, enter "10008".

Note: The start date will default to the current date.  Update this date as necessary.


To update the date, click the Project Start Date field.

Enter the appropriate project starting date into the Project Start Date field.


In this example, enter "5/1/17".

Enter the project ending date into the Project Finish Date field.


In this example, enter "12/31/20".

To update the default organization, click the Organization drop-down list.

Click to select the appropriate organization.


In this example, click Global Information Technologies.

Click to select the appropriate legal entity.


In this example, click KBACE Technologies, Inc..

The Search and Select: Project Manager window is displayed.


Enter all or part of the project manager's name into the Name field.


In this example, enter "Marlin".

Enter any additional search criteria as needed.


To execute the search based on the criteria entered, click the Search button.

Names matching the search criteria are listed.


Double-click to select the appropriate project manager name.


In this example, double-click Marlin, Amy.

Click the scrollbar to navigate to the Copy Options section.

Click the scrollbar to navigate to the Financial Plans section.

Click the scrollbar to navigate up to the Save and Continue button.

To continue, click the Save and Continue button.

The Manage Financial Project Settings page is displayed.


The project definitions are addressed in detail on the referenced tabs of this worksheet.


Click the scrollbar to view additional details.

Project general details are displayed.


Click the Parties tab.

All valid team members, project customers, partner organizations and supplier organizations are listed.


Click the Planning tab.

Additional project planning details are displayed.


Click the Financial tab.

Project financial details are displayed.


Click the Reporting tab.

Project reporting details are displayed.


To continue, click the Done button.

To complete the process, click the Done button.

The Organization and Legal Entity default based on the selected template. The fields may be updated as needed.

The Quick Entry fields defined on a project template control minimum required information to create a project using that template.


For example, apart from the default required fields such as Project Name and Project Number, the project template may require that you specify a project manager and project administrator, or a funding source.

Use the checkboxes next to the Project Information and Project Plan Information fields to copy the information from the selected template.


In this example, accept the defaults.

Use the checkboxes next to the Financial Plans and Other Financial Information fields to copy the information from the selected template.


In this example, accept the defaults.

The Overhead Project has been created.
