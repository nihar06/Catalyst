This topic covered:


- Approving a Project Budget

The Oracle Home page is displayed.


Click the Projects button.

The Projects window is displayed.


Click the Project Financial Management button.

The My Projects page is displayed.


Click the Project Number link corresponding to the project budget to be approved.


In this example, click C10009.

The corresponding Overview page is displayed.


Click the Tasks button.

The Tasks pane is displayed.


In the Control section, click Manage Project Budget.

The corresponding Manage Budget Versions page is displayed.


Click to select the version Name cell corresponding to the version to be reviewed and approved.


In this example, click Version 2.

Click Review and Finalize Budget.

Note: If the budget is ready to send for approval use the Submit drop-down menu and then click the Create Baseline command.


The corresponding Submit Version message window is displayed.


To continue, click the OK button.

From the Overview page, click the Done button.

The budget has been successfully approved.
