This topic covered:


- Editing a Classification on a Project

The Oracle Home page is displayed.


The Projects window is displayed.


The My Projects page is displayed.


To expand the search area, click the Expand Search button.

To search for the project by project number, click the Project Number field.

Enter all or part of the project number into the Project Number field.


In this example, enter "D10000".

To include project status as part of the search, click the Project Status drop-down list.

To include all statuses in the search, click the All list item.

Enter any additional search criteria as needed.


To execute the search based on the criteria entered, click the Search button.

Project numbers matching the search criteria are listed.


Click to select the appropriate project number link


In this example, click the D10000 link.

The corresponding Overview page is displayed.


The Tasks list is displayed.


In the Initiate section, click the Manage Financial Project Settings link.

The corresponding Manage Financial Project Settings page is displayed.


The Edit Project Classification window is displayed.


In the new row, click the Class Category drop-down list.

Click to select the appropriate class category from the list.


In this example, click Business Objective.

Click to select the appropriate class code from the list.


In this example, click Business Growth.

To continue, click the Save and Close button.

To complete the process, click the Done button.

From the Overview page, click the Done button.

The classification has been updated for the selected project.
