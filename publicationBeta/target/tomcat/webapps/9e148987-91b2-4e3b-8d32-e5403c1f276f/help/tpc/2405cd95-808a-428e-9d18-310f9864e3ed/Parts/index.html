Print It! mode displays a document supplied for the topic, if available. This is typically a job aid, system document, or some other document with pertinent information on the topic. The document will launch in either Microsoft Word, Adobe Acrobat, or in your browser (HTML) depending on the format of the document provided. You can review the document online or print it.

 
Note: You can also use Print It! from the options on Actions link in the bubble for the play modes that support it.
